Goldendoodle Quiz



How I Addressed the Project

I utilized my skills in HTML and Javascript to create a quiz on Goldendoodles. The subject matter is close to heart, as I have two Goldendoodle puppies, Rip and Ralphie.



What I Created

This project is composed of HTML, Javascript and CSS. I created this project by significantly modifying a template my professor provided.

Background color and image: I decided on a background color (#d8bf9f), but I wanted to take the background design a step further. I created a background image in Canva, then exported it as a SVG. I placed the .SVG in the web files folder and linked to it in the head of the page. I gave the image certain attributes so that it would appear as I intended, such as not repeating the image, ensuring the image is fixed in the center of the page and covers the screen. Since the image is an SVG, this file type can withstand those attributes due to its ability to resize at any viewpoint. Detailed code changes included below:
background-image: url('goldendoodlebackground.svg');
background-repeat: no-repeat;
background-attachment: fixed;
background-size: cover;
background-position: center;

The background image includes pictures of my two Goldendoodles, Rip and Ralphie, both 1.5 years old. They are such sweet boys and love to pose for pictures (treats).

Quiz background: I gave the quiz a white background with rounded edges to create a crisp contrast with the background image. I also wanted to ensure the quiz did not overlap the areas of the background image with the dogs, so I set the width to 50% and ensured the content was center aligned.Further, I ensured the div was center aligned.

Text: In head, I linked to a few Google Fonts I utilized in this quiz: Nunito for headings and DM Sans for body text. I updated the heading text to dark brown to fit the golden theme and create an accessible text contrast ratio from the background image. I ensured all text met WCAG 2.1 AA standards for accessibility using Adobe Color's accessibility tools. I also added padding throughout the page. For instance, the h1 element has padding-top:30px. Further, I added a line break in the text directly under the heading by adding a line break. Buttons: I updated the color, shape and font of the previous and next buttons.

Quiz Functionality (Javascript): I updated the existing code with five of my quiz questions, then added an additional five questions. To add the additional five questions, I had to first update var total=10, then add five additional questions with their corresponding answer choices and solutions. Then, I added my seven answer feedback options.





What I Learned

This was my first project coding by hand rather than working with a web builder such as WebFlow or Wix. Therefore, there was a learning curve. However, I was able to connect to and build upon my previous knowledge of basic HTML, CSS and Javascript. This was a rewarding project, as I now feel much more confident in my coding abilities in these languages.

Game of Thrones Drag-and-Drop Quiz



How I Addressed the Project

I utilized my skills in HTML and CSS to create a drag-and-drop quiz based on Game of Thrones. Learners can select a character from the list, then drag and drop the character to their respective family house.



What I Created

This project is composed of HTML and CSS. I created this project by significantly modifying a template my professor provided.

Below is a description of the revisions I made to both the HTML and CSS files.

HTML
I added a Google font embed for utilized fonts (I embedded a few but only used Nanum Myeongjo). I adjusted the title and instructions.
I revised the existing boxes by adding each house's name. Then, I adjusted the existing drag and drop tiles by adding the corresponding character names.
I added boxes 4 and 5, along with their corresponding character names.
I adjusted the correct pop-up message to read “Congratulations! Everything is correct. But remember, the night is dark and full of terrors.”

CSS
I adjusted the font in {body} to Nanum Myeongjo (Google Fonts, embed in html).
I downloaded an open source image of Scotland (simliar landscape to some areas in Westeros) and added it to the web files folder. Then, I added it as a background image in {body} and added properties to it. Below is a code snippet:
background-image: url("scotland.jpg");
background-position: center;
background-repeat: no-repeat;
background-size: cover;
position: relative;

Box borders: I changed the border color to black (#000000) and rounded only the bottom two corners of the elements by 15px. I was able to round only two corners through the following:
border-radius: 0px 0px 15px 15px;

I adjusted the “house” box headings to a deep red (#412926).
I adjusted the “character” box heading to a deep charcoal color (#212026).
I set the main container as white with a 90% transparency. I achieved this through the following:
background-color: rgba(255, 255, 255, 0.9)
I set the correct color to: #115930. I ensured that this color met WCAG 2.1 AA standards for text-contrast ratio.
I set the incorrect color to: #D13D4B. I ensured that this color met WCAG 2.1 AA standards for text-contrast ratio.
I set the background color of the movable items to: #C1B990.
I adjusted the height of the main container and left (sources/character) column to fit the content better.

What I Learned

I created this project after the Goldendoodle quiz, so I was able to build upon my skills in HTML and CSS. In this project, I was able to use more advanced styling, such as element transparency and corner rounding. A challenge I experienced during development was the CSS being a standalone file. This meant that I had to be careful in identifying elements that needed revision, as I would accidentally edit the wrong element. By learning this hands-on and making mistakes, I developed a better understanding of how HTML and CSS files interact. Further, I learned how to be precise and calculated when coding by hand.
